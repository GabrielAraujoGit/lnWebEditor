## <img src="https://api.iconify.design/lucide/info.svg" width="18" /> Descrição Geral


O **lnWebEditor** é um editor web leve e independente, desenvolvido para
leitura, execução e análise de **scripts LN/Baan** diretamente no
navegador --- sem instalação, sem dependências externas e totalmente
*offline-ready*.

Ele permite que equipes técnicas **testem, debuguem e validem scripts LN
sem instalar o LN Studio**, aumentando significativamente a
produtividade.

O projeto roda 100% no navegador por meio de **HTML + CSS +
JavaScript**, garantindo portabilidade máxima.

------------------------------------------------------------------------

## <img src="https://api.iconify.design/lucide/list.svg" width="18" /> Funcionalidades


-   **Syntax Highlight completo para LN/Baan**
-   **Execução simulada de scripts (sandbox)**
-   **Debug e análise rápida sem LN Studio**
-   **Interface responsiva e minimalista**
-   **Zero dependências externas**
-   **Funciona offline (somente abrir o index.html)**

## <img src="https://api.iconify.design/lucide/play.svg" width="18" /> Como Usar


### 1. Clonar o repositório

``` bash
git clone https://github.com/GabrielAraujoGit/lnWebEditor.git
```

### 2. Abrir o editor

Basta abrir o arquivo:

    index.html

Nenhuma instalação é necessária. O editor funciona totalmente offline.
